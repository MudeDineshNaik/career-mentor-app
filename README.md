# AI Career Mentor

This is a career-roadmap generation tool built for students to convert their background into a 30-day skill learning plan.

## How to Run This Locally
1. Clone this repository: `git clone https://github.com/MudeDineshNaik/career-mentor-app.git`
2. Install the requirements: `pip install -r requirements.txt`
3. Create a `.env` file and add your Gemini API Key: `GEMINI_API_KEY=your_key_here`
4. Run the app: `streamlit run app.py`

## Tech Stack
- Python
- Streamlit
- Google Gemini API
